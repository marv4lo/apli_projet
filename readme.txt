

Pour utiliser l’application du dossier Appli_gestion_stages, il faut créer une base de donnée “entreprises” dans laquelle on exporte le fichier entreprise.sql qui se trouve dans la library.
Il faut également entrer votre username et votre password pour se connecter à la base de données dans le fichier database.php qui se trouve dans le répertoire classes.
